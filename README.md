# SlideTools Console Application

<img src="Images/2026012801-01a.jpg" width="100%" alt="14th century monks working at a document automation contraption" />

This is a command-line application specialized in working with PowerPoint documents.

<p>&nbsp;</p>

## Features

Following is the current list of available features in this version.

-   **DistributeHorizontally**. Distribute all selected items between the leftmost and rightmost items.
-   **DistributeVertically**. Distribute all selected items between the topmost and bottom-most items.
-   **FindObjects**. Find and select all objects matching the caller-specified filter elements.
-   **ForEachSlide**. Loop through each slide in the Presentation property of the local working document, running the defined child actions in the context of each slide.
-   **GetSelectedMaxY**. Assign the maximum Y coordinate of the selected objects to the specified variable.
-   **GetSelectedMaxYItem**. Assign the reference of the item having the maximum Y coordinate of all selected objects to the specified variable.
-   **GetSelectedMinX**. Assign the minimum X coordinate of the selected objects to the specified variable.
-   **GetSelectedMinY**. Assign the minimum Y coordinate of the selected objects to the specified variable.
-   **GetSelectedMinYItem**. Assign the reference of the item having the minimum Y coordinate of all selected objects to the specified variable.
-   **SetItemYFromVariable**. Assign the Y coordinate of the specified item from the value in the specified variable.
-   **SetSelectedMaxWidth**. Set the maximum width of all items in the selection.
-   **SetSelectedX**. Assign the X coordinate on all selected objects to the value in the specified variable.
-   **SlideReport**. Create a report of all the slides in the specified file.

<p>&nbsp;</p>

Following is the list of features currently under construction.

-   **AlignLeft**. Align all selected items left.
-   **ChangeImage**. Switch the image on all selected items.
-   **ForEachSelected**. Loop through each item in the selected items list, performing a set of actions on each selected item.
